# Bash and Linux basic commands

- where am i
```
pwd
```
- How to create a Dir
```
mkdir name_of_folder
```

- How to create a file
```
touch name_of_file.md
```

- How to change dir
```
cd dir_name/
```

- How to go out of dir
```
cd ..
```

- Who am i
```
uname -srm
```

- list dir and all dir
``` ls ``` or ``` ls -a ```

- Move or rename a file
```
mv current_file new_file_or_location
```

- force delete
```
rm -rf name_of_file
```
